注册/登录模块实现步骤说明

数据库准备
使用MySQL Workbench（图形化工具）

步骤1：打开MySQL Workbench
1. 启动MySQL Workbench
2. 连接到MySQL服务器

步骤2：打开SQL文件
1. 点击菜单：`File` -> `Open SQL Script...`
2. 浏览到项目目录，选择 `database/schema.sql` 文件
3. 文件内容会显示在编辑器中

步骤3：执行SQL语句
点击工具栏上的执行按钮

步骤4：验证结果
在左侧导航栏中，点击刷新按钮

步骤5：配置数据库连接
编辑 `src/main/resources/application-dev.yml`：
```yaml
spring:
  datasource:
    username: root
    password: your_password
```

步骤6：运行项目
在IDE中右键 `CertificationApplication.java` -> Run
